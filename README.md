# Gambling Game

## Project Structure

```
Gambling-Game/
├── src/
│   ├── main.py            # Main game logic
│   └── utils.py           # Helper functions
├── tests/                  # Test cases
│   └── test_main.py       # Tests for main game logic
├── README.md               # Project overview
└── requirements.txt        # Dependencies
```

## Description

This project is a gambling game where users can play different gambling-related games. The goal is to provide a fun and interactive experience while educating users about probabilities and strategies.

## Features
- Multiple game types (e.g., Blackjack, Poker)
- User account management
- Betting system
- Interactive user interface

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/krisjscott/Gambling-Game.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Gambling-Game
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the main game:
   ```bash
   python src/main.py
   ```
